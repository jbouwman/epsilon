# Binding Intermediate Representation (BIR) Format

BIR is a persistent, human-readable S-expression format for storing parsed C header information.

## Format Version

Current version: **1**

The version field enables future format changes with migration support.

## File Structure

```lisp
(:binding-ir
 :version 1
 :source-headers ("header1.h" "header2.h")
 :include-paths ("/usr/include" "/usr/local/include")
 :defines (("DEBUG" . "1"))
 :parse-time 3940502400
 :platform :linux-x86-64

 :functions (...)
 :structs (...)
 :enums (...)
 :typedefs (...)
 :macros (...))
```

## Top-Level Fields

| Field | Type | Description |
|-------|------|-------------|
| `:version` | integer | BIR format version |
| `:source-headers` | list | Header files that were parsed |
| `:include-paths` | list | Include directories used |
| `:defines` | alist | Preprocessor definitions used |
| `:parse-time` | integer | Universal time when parsed |
| `:platform` | keyword | Target platform (e.g., `:linux-x86-64`) |

## Function Entries

```lisp
(:name "function_name"
 :return-type :int
 :params ((:name "arg1" :type :pointer)
          (:name "arg2" :type :double))
 :variadic-p nil
 :header "source.h"
 :line 123)
```

| Field | Type | Description |
|-------|------|-------------|
| `:name` | string | C function name |
| `:return-type` | keyword | Return type (`:int`, `:pointer`, etc.) |
| `:params` | list | Parameter list |
| `:variadic-p` | boolean | True if variadic (`...`) |
| `:header` | string | Source header file |
| `:line` | integer | Line number in header |

## Struct Entries

```lisp
(:name "struct_name"
 :size 24
 :alignment 8
 :fields ((:name "field1" :type :int :offset 0 :size 4)
          (:name "field2" :type :pointer :offset 8 :size 8))
 :opaque-p nil)
```

| Field | Type | Description |
|-------|------|-------------|
| `:name` | string | Struct name |
| `:size` | integer | Total size in bytes |
| `:alignment` | integer | Alignment requirement |
| `:fields` | list | Field definitions |
| `:opaque-p` | boolean | True if forward-declared only |

## Enum Entries

```lisp
(:name "enum_name"
 :values ((:name "VALUE_A" :value 0)
          (:name "VALUE_B" :value 1)
          (:name "VALUE_C" :value 42)))
```

## Typedef Entries

```lisp
(:name "my_type"
 :underlying "unsigned int")
```

## Macro Entries

```lisp
(:name "MACRO_NAME"
 :kind :constant
 :value 42)

(:name "FUNC_MACRO"
 :kind :function-wrapper
 :target "actual_function")
```

## Type Keywords

Standard C types map to keywords:

| C Type | BIR Keyword |
|--------|-------------|
| `void` | `:void` |
| `char` | `:char` |
| `short` | `:short` |
| `int` | `:int` |
| `long` | `:long` |
| `long long` | `:long-long` |
| `float` | `:float` |
| `double` | `:double` |
| `unsigned int` | `:unsigned-int` |
| `unsigned long` | `:unsigned-long` |
| `size_t` | `:size-t` |
| `T*` (any pointer) | `:pointer` |

## Example BIR File

```lisp
;; Generated by epsilon.foreign.binding-ir
;; Source: /usr/include/string.h
;; Time: 2026-02-02T10:30:00

(:binding-ir
 :version 1
 :source-headers ("/usr/include/string.h")
 :parse-time 3940502400
 :platform :linux-x86-64

 :functions
 ((:name "strlen"
   :return-type :size-t
   :params ((:name "s" :type :pointer))
   :variadic-p nil
   :header "string.h"
   :line 384)

  (:name "strcpy"
   :return-type :pointer
   :params ((:name "dest" :type :pointer)
            (:name "src" :type :pointer))
   :variadic-p nil
   :header "string.h"
   :line 121))

 :structs ()
 :enums ()
 :typedefs ()
 :macros ())
```

## Validation

BIR files are validated on load:

1. Magic/version check
2. Required fields present
3. Type keywords recognized
4. Cross-references valid

Use `validate-binding-ir` to check programmatically:

```lisp
(epsilon.foreign.binding-ir:validate-binding-ir ir)
;; Returns t or signals error with details
```

## Compatibility

Use `bir-compatible-p` to check if a BIR structure is compatible:

```lisp
(epsilon.foreign.binding-ir:bir-compatible-p ir)
;; Returns t if version is supported
```
